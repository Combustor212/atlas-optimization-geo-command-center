# AGS - Local SEO & AI Visibility Platform

A standalone Vite+React application for local SEO analysis, AI visibility checks, and business scanning.

## Features

- **Smart Business Scanner**: Google Places API integration with autocomplete suggestions
- **AI Visibility Analysis**: OpenAI ChatGPT integration for geo-profile scanning
- **Local SEO Tools**: Comprehensive visibility scoring and recommendations
- **Content Management**: FAQ, Reviews, and SEO settings management

## API Keys

The application uses the following API keys (configured in `src/config/apiKeys.js`):
- OpenAI ChatGPT API: For AI visibility analysis
- Google Places API: For business search and autocomplete

## Running the app

```bash
npm install
npm run dev
```

## Building the app

```bash
npm run build
```

## Technology Stack

- React 18
- Vite
- Tailwind CSS
- React Router
- Framer Motion
